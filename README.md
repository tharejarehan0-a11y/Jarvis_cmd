# Jarvis
a simple cmd based talking bot with gemini api 
1. This performs simple functions like listening what you speak
2. then giving the text to gemini
3. then responding by voice using the speech recoginition module
if you want to exit the loop then just say exit

# How to run
1. go to the dist folder
2. In the dist folder go to the jarvis.exe file 
3. Click on view raw link and download the file 
4. Double click on the file to run it
