Jarvis
a simple cmd based talking bot with gemini api

. This performs simple functions like listening what you speak
. then giving the text to gemini
. then responding by voice using the speech recoginition module 
when you want to exit the loop then just say exit

How to run
. go to the build folder
. In the build folder go to the jarvis.exe file
. Click on view raw link and download the file
. Double click on the file to run it
